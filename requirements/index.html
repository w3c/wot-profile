Use Cases

Smart Buildings

As a developer of a smart building hub, I want to safely expose a standardised web-based API for monitoring and controlling a range of IoT devices in a commercial building using different underlying protocols, so that cloud services from multiple vendors can consume data from that hub without having to implement a vendor-specific API.

As a developer of a smart building cloud service, I want to be able to provide data analytics for smart buildings using smart building hubs from different vendors which conform to a common standard, rather than having to implement multiple vendor-specific APIs.
